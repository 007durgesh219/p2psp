# Step 2: Install Python 3
<!-- {{{  -->

## Windows

<!-- {{{  -->

1. Install (as Administrator) [Python 3](https://www.python.org/downloads).

<!-- }}} -->

## Mac

<!-- {{{  -->

1. Install [Python 3](https://www.python.org/downloads/mac-osx/).

<!-- }}} -->

## Ubuntu Linux

<!-- {{{  -->

Run:

   '''
   sudo apt-get update
   sudo apt-get install python3
   '''

<!-- }}} -->

## Arch Linux

<!-- {{{  -->

Run:

   '''
   sudo pacman -Sy
   sudo pacman -S python3
   '''

<!-- }}} -->

<!-- }}} -->
